# projeto-java-springboot-api_Rest
Projeto Spring Boot focado no desenvolvimento de uma API REST, abordando conceitos como Controllers, DTOs, Java Records, validações personalizadas com Bean Validation (ConstraintValidator), filtros via parâmetros de requisição e boas práticas no desenvolvimento de serviços backend.
